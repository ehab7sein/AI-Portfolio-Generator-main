## Instructions for Use:

1. Open each file (data-collection.html, design-selection.html)
2. Use Find & Replace (Ctrl+H) in your editor
3. Search for the Arabic text and replace with English equivalent
4. Make sure to check:
   - HTML content between tags
   - Placeholder attributes
   - JavaScript strings
   - Alert/notification messages
   - Form validation messages

## Note:
Some texts may need contextual adjustment. Review each replacement to ensure it makes sense in context.
